# Wanted Posters

Every wanted poster below is generated by old "Ten Mile" Cooter Tunk back at CI from configs defined in
[`scripts/screenshots/configs.lua`](scripts/screenshots/configs.lua). Each
image is labeled with its **config key** (e.g. `popupmenu_border`, `hl_neon`,
`theme_saloon`).

If something looks off, [open an issue](https://github.com/catgoose/wildest.nvim/issues)
and reference the config key so we can reproduce it.
